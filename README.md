# inkpot_gc_gtk

## Description
Gynvael Coldwind's inkpot_gc theme for GVim/Sublime ported to GTK+ editors.
This is only theme, not full config. Based on screenshots from stream. Some
colors were taken directly from inkpot_gc.vim. Others may be a little vary
in comparison to GVim/Sublime original due the optimization for gedit.
